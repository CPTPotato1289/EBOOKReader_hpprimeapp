# EBOOKReader_hpprimeapp
这是一个功能简单但好用的运行于HPPrime计算器上的电子书阅读器
不仅支持文本格式的内容，更能在书本中插入图片！
使用python编写

# 使用方法
## 1 准备书本文件
1. 首先，将电子书转换为txt格式，文件名为`bookname_book.txt`
2. 将分章节的地方添加章节表示，格式为`[ch?()章节名称]`（`？`从0开始）。这个标记符必须独立成一行
3. 在需要插入图片的地方单独成行写图片标识`[pic:imagine_name.jpg]`。支持jpg和png格式。注意图片需要压缩，分辨率不可过高。
4. 在与txt同目录下放置txt中引用的所有图片，文件名许对应
5. 注意：单个txt文件大小最大约为0.5MB，更大可能会爆内存。如果爆内存请分册存储。
## 2 上传到计算器
1. clone本仓库
2. 将仓库内的EBOOKReader.hpappdir去除，将刚才制作好的所有书本文件全部防止到该目录下。注意不要新建子目录！
3. 使用HP Connectivity Kit将EBOOKReader.hpappdir上传到计算器
## 3 如何使用
1. 在计算器Apps视图选择EBookReader应用。
2. 按下Num键，点击屏幕上清除
3. 根据提示进行阅读即可
4. 注意：每本书必须先构建索引才能阅读。修改设置中字体大小时需要重建索引。
5. 使用下、右、enter翻到下一页，使用左、上翻到上一页。
6. 图片页可用2，4，6，8键移动，+-键放大缩小图片；也可以用手指在屏幕上滑动移动图片。
7. 如果遇到内存不足报错尝试按下shift+plot修改Heap为更大的值
